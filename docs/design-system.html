Settings

Active = filled icon + sand-900. Inactive = outline icon + sand-500. Double cue (shape + colour) for accessibility.

Accessibility

Contrast minimums

sand-700 on sand-50 4.8:1 ratio — AA pass
sand-700 on sand-100 4.5:1 ratio — AA pass
sand-100 on sand-900 15.2:1 ratio — AAA pass

Rules

Never use sand-500 or lighter for body text. Minimum is sand-700 (#7A5C3A). Viewport allows user zoom. All icon buttons must have aria-label. Touch targets minimum 44x44px.
